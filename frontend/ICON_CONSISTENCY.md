# Иконка приложения - единообразие

## ✅ Гарантия единообразия иконки

Все иконки генерируются из одного источника - `butterfly.svg`. Это гарантирует, что приложение будет иметь одинаковую иконку во всех случаях:

- ✅ При добавлении на главный экран iOS
- ✅ При установке как PWA на Android
- ✅ В браузере (favicon)
- ✅ В списке приложений
- ✅ На главном экране устройства

## Генерация иконок

Запустите скрипт для генерации всех необходимых размеров:

```bash
cd frontend
npm run generate-icons
```

Или вручную:
```bash
bash scripts/generate-icons.sh
```

## Структура иконок

### Источник
- `public/butterfly.svg` - исходная иконка

### Генерируемые размеры

**Favicons (браузер):**
- 16x16, 32x32, 96x96

**iOS/Apple Touch Icons:**
- 57x57, 60x60, 72x72, 76x76, 114x114, 120x120, 144x144, 152x152, 180x180
- `apple-touch-icon.png` (180x180) - основная иконка для iOS

**Android/PWA:**
- 128x128, 192x192, 384x384, 512x512

**Maskable (Android адаптивные иконки):**
- 192x192-maskable, 512x512-maskable (с отступами для адаптивных иконок)

## Проверка единообразия

После генерации иконок проверьте:

1. **В браузере:**
   - Откройте DevTools → Application → Manifest
   - Проверьте, что все иконки загружаются без ошибок

2. **iOS (Safari):**
   - Откройте сайт в Safari
   - Нажмите "Поделиться" → "На экран «Домой»"
   - Проверьте, что иконка отображается корректно

3. **Android (Chrome):**
   - Откройте сайт в Chrome
   - Установите как PWA
   - Проверьте иконку на главном экране

4. **Визуальная проверка:**
   - Все иконки должны выглядеть одинаково (только размер разный)
   - Иконка должна быть узнаваемой во всех размерах

## Важно

- ❌ **НЕ** редактируйте отдельные PNG файлы вручную
- ✅ Всегда используйте `butterfly.svg` как источник
- ✅ Регенерируйте все иконки после изменения `butterfly.svg`
- ✅ Проверяйте единообразие после генерации

## Troubleshooting

### Иконка отличается на разных платформах

1. Убедитесь, что все иконки сгенерированы из одного источника
2. Очистите кэш браузера и устройства
3. Переустановите PWA
4. Проверьте, что `manifest.json` ссылается на правильные файлы

### Иконка не обновляется

1. Очистите кэш: Settings → Apps → Ваше приложение → Clear cache
2. Переустановите PWA
3. Проверьте, что файлы действительно обновились на сервере

### Иконка размытая

1. Убедитесь, что иконки сгенерированы в правильных размерах
2. Используйте PNG (не JPEG) для иконок
3. Проверьте, что исходный SVG имеет достаточное качество
